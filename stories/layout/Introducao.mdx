# Nephos Layout

O **Nephos Layout** é um bloco completo de layout e estrutura de página, fornecendo componentes prontos para construção de interfaces administrativas.

## 📦 Instalação

```bash
npm install @iatec/nephos-layout
```

## 🎯 Características

- ✅ Sidebar responsivo com navegação
- ✅ Topbar customizável
- ✅ Profile Sidebar para perfil de usuário
- ✅ Suporte a temas claro e escuro
- ✅ Configurador de layout integrado
- ✅ Totalmente compatível com PrimeNG
- ✅ Responsivo e mobile-first

## 📋 Componentes Disponíveis

### Layouts

- **LayoutComponent** (`nph-layout`): Layout completo com sidebar

### Componentes

- **SidebarComponent**: Sidebar de navegação lateral
- **TopbarComponent**: Barra superior de navegação
- **ProfileSidebarComponent**: Sidebar para perfil do usuário
- **MenuComponent**: Menu de navegação
- **BreadcrumbComponent**: Navegação breadcrumb
- **LayoutConfigurator**: Configurador visual do layout

## 🚀 Uso Básico

### 1. Importar o Módulo

```typescript
import { LayoutComponent } from '@iatec/nephos-layout';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [LayoutComponent],
  template: `<nph-layout />`
})
export class AppComponent {}
```

### 2. Importar os Estilos

No seu arquivo `styles.scss`:

```scss
@import '@iatec/nephos-layout/lib/layout/layout.scss';
```

### 3. Exemplo Completo

```typescript
import { Component } from '@angular/core';
import { RouterOutlet } from '@angular/router';
import { LayoutComponent } from '@iatec/nephos-layout';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [RouterOutlet, LayoutComponent],
  template: `
    <nph-layout>
      <ng-template #logo>
        <img src="assets/logo.svg" alt="Logo" />
      </ng-template>
      
      <ng-template #topbar>
        <!-- Conteúdo customizado do topbar -->
      </ng-template>
      
      <ng-template #profileSidebar>
        <!-- Conteúdo do perfil -->
      </ng-template>
    </nph-layout>
  `
})
export class AppComponent {}
```

## 🎨 Temas

O layout suporta múltiplos temas de menu:

- **colorScheme**: Usa o esquema de cores do tema
- **primaryColor**: Usa a cor primária
- **transparent**: Menu transparente

## 📱 Responsividade

O layout é totalmente responsivo e se adapta automaticamente para:

- **Desktop** (> 992px): Layout completo
- **Tablet** (768px - 992px): Layout adaptado
- **Mobile** (< 768px): Menu colapsável

## 🔧 Serviços

### LayoutService

Serviço principal para controle do layout:

```typescript
import { LayoutService } from '@iatec/nephos-layout';

constructor(private layoutService: LayoutService) {
  // Alterar modo do menu
  this.layoutService.updateLayoutConfig({ menuMode: 'slim' });
  
  // Alternar tema
  this.layoutService.updateLayoutConfig({ darkTheme: true });
  
  // Toggle sidebar
  this.layoutService.toggleSidebar();
}
```
