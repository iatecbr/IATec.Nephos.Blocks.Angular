# Configuração do Nephos Layout

Guia completo de configuração e personalização do Nephos Layout.

## 🔧 LayoutService

O `LayoutService` é o serviço principal para controlar e configurar o layout.

### Injetando o Serviço

```typescript
import { LayoutService } from '@iatec/nephos-layout';

constructor(private layoutService: LayoutService) {}
```

## ⚙️ Configurações Disponíveis

### Layout Config (layoutConfig)

Interface `LayoutConfig`:

```typescript
interface LayoutConfig {
  ripple: boolean;              // Efeito ripple nos componentes
  inputStyle: 'outlined' | 'filled';  // Estilo dos inputs
  menuMode: MenuMode;           // Modo do menu
  menuTheme: MenuTheme;         // Tema do menu
  colorScheme: ColorScheme;     // Esquema de cores
  theme: string;                // Nome do tema
  scale: number;                // Escala da interface (12-16)
}
```

### Modos de Menu (MenuMode)

```typescript
type MenuMode = 
  | 'static'      // Fixo sempre visível
  | 'overlay'     // Sobrepõe o conteúdo
  | 'slim'        // Compacto com expansão
  | 'slim-plus'   // Compacto com ícones e labels
  | 'horizontal'  // Menu horizontal no topo
  | 'reveal'      // Oculto que revela no hover
  | 'drawer';     // Gaveta lateral
```

### Temas de Menu (MenuTheme)

```typescript
type MenuTheme = 
  | 'colorScheme'   // Usa o esquema de cores
  | 'primaryColor'  // Usa a cor primária
  | 'transparent';  // Menu transparente
```

### Esquema de Cores (ColorScheme)

```typescript
type ColorScheme = 
  | 'light'  // Tema claro
  | 'dark'   // Tema escuro
  | 'dim';   // Tema intermediário
```

## 📝 Exemplos de Uso

### Alterar Modo do Menu

```typescript
// Modo Static
this.layoutService.updateLayoutConfig({ 
  menuMode: 'static' 
});

// Modo Overlay
this.layoutService.updateLayoutConfig({ 
  menuMode: 'overlay' 
});

// Modo Slim
this.layoutService.updateLayoutConfig({ 
  menuMode: 'slim' 
});
```

### Alternar Tema

```typescript
// Ativar tema escuro
this.layoutService.updateLayoutConfig({ 
  colorScheme: 'dark' 
});

// Voltar para tema claro
this.layoutService.updateLayoutConfig({ 
  colorScheme: 'light' 
});
```

### Alterar Tema do Menu

```typescript
// Menu com cor primária
this.layoutService.updateLayoutConfig({ 
  menuTheme: 'primaryColor' 
});

// Menu com esquema de cores
this.layoutService.updateLayoutConfig({ 
  menuTheme: 'colorScheme' 
});

// Menu transparente
this.layoutService.updateLayoutConfig({ 
  menuTheme: 'transparent' 
});
```

### Alterar Escala da Interface

```typescript
// Aumentar tamanho da interface
this.layoutService.updateLayoutConfig({ 
  scale: 14 
});

// Tamanho padrão
this.layoutService.updateLayoutConfig({ 
  scale: 12 
});
```

### Configuração Completa

```typescript
this.layoutService.updateLayoutConfig({
  ripple: true,
  inputStyle: 'outlined',
  menuMode: 'static',
  menuTheme: 'colorScheme',
  colorScheme: 'light',
  theme: 'indigo',
  scale: 12
});
```

## 🎛️ Layout State

O `layoutState` controla o estado dinâmico do layout:

```typescript
interface LayoutState {
  staticMenuDesktopInactive: boolean;  // Menu desktop inativo
  overlayMenuActive: boolean;          // Menu overlay ativo
  profileSidebarVisible: boolean;      // Sidebar de perfil visível
  configSidebarVisible: boolean;       // Sidebar de config visível
  staticMenuMobileActive: boolean;     // Menu mobile ativo
  menuHoverActive: boolean;            // Hover do menu ativo
  sidebarActive: boolean;              // Sidebar ativo
  anchored: boolean;                   // Sidebar ancorado
}
```

### Manipulando o State

```typescript
// Toggle do sidebar
this.layoutService.toggleSidebar();

// Mostrar profile sidebar
this.layoutService.showProfileSidebar();

// Esconder profile sidebar
this.layoutService.hideProfileSidebar();

// Toggle configurador
this.layoutService.showConfigSidebar();
```

### Observables

```typescript
// Observar mudanças de configuração
this.layoutService.config$.subscribe(config => {
  console.log('Nova configuração:', config);
});

// Observar estado do layout
this.layoutService.state$.subscribe(state => {
  console.log('Novo estado:', state);
});

// Observar abertura do overlay
this.layoutService.overlayOpen$.subscribe(() => {
  console.log('Menu overlay aberto');
});
```

## 📦 Configuração Inicial

### No AppComponent

```typescript
import { Component, OnInit } from '@angular/core';
import { LayoutService } from '@iatec/nephos-layout';

@Component({
  selector: 'app-root',
  template: '<nph-layout></nph-layout>'
})
export class AppComponent implements OnInit {
  
  constructor(private layoutService: LayoutService) {}
  
  ngOnInit() {
    // Configuração inicial do layout
    this.layoutService.updateLayoutConfig({
      ripple: true,
      inputStyle: 'outlined',
      menuMode: 'static',
      menuTheme: 'colorScheme',
      colorScheme: 'light',
      theme: 'indigo',
      scale: 12
    });
    
    // Carregar preferências do usuário do localStorage
    const savedConfig = localStorage.getItem('layout-config');
    if (savedConfig) {
      this.layoutService.updateLayoutConfig(JSON.parse(savedConfig));
    }
    
    // Salvar mudanças no localStorage
    this.layoutService.config$.subscribe(config => {
      localStorage.setItem('layout-config', JSON.stringify(config));
    });
  }
}
```

```typescript
// Observar mudanças de tamanho
this.layoutService.state$.subscribe(state => {

  } else if (this.layoutService.isDesktop()) {
    console.log('Modo desktop');
  }

```

## 🚀 Dicas de Performance

1. **Lazy Loading**: Use lazy loading para módulos de rotas
2. **OnPush Strategy**: Use ChangeDetectionStrategy.OnPush em componentes
3. **Unsubscribe**: Sempre faça unsubscribe de observables
4. **Virtual Scroll**: Use virtual scrolling para listas grandes no menu
