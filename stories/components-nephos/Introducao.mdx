# Blocos Nephos

Componentes customizados e blocos de interface completos desenvolvidos internamente pela equipe Nephos.

## 🧱 O que são Blocos?

Blocos são componentes de alto nível que combinam múltiplos elementos para formar funcionalidades completas. Diferente dos componentes básicos do PrimeNG, os blocos são:

- **Específicos do Nephos**: Desenvolvidos internamente para necessidades específicas
- **Compostos**: Combinam múltiplos componentes PrimeNG e customizados
- **Completos**: Incluem lógica, estilo e comportamento integrados
- **Documentados**: Cada bloco tem documentação completa de uso

## 📦 Estrutura dos Blocos

Os blocos estão organizados nas bibliotecas:

```
projects/iatec/
├── nephos-layout/    # Blocos de layout e estrutura
├── nephos-pages/     # Páginas e templates completos
└── nephos-utils/     # Utilitários e helpers
```

## 🔧 Importação Geral

Para usar os blocos, importe as bibliotecas necessárias:

```typescript
// Layout components
import { NephosLayoutModule } from '@iatec/nephos-layout';

// Page templates
import { NephosPagesModule } from '@iatec/nephos-pages';

// Utilities
import { NephosUtilsModule } from '@iatec/nephos-utils';

@NgModule({
  imports: [
    NephosLayoutModule,
    NephosPagesModule,
    NephosUtilsModule
  ]
})
export class AppModule { }
```
